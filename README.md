# AI-Powered Business Analytics & Decision Intelligence Platform

## Overview

An AI-powered analytics platform built using Python, Streamlit, Pandas, Scikit-Learn and Google Gemini AI.

## Features

* CSV Upload
* KPI Dashboard
* Region Filter
* Category Filter
* Monthly Sales Trend
* Top Customers Analysis
* Top Products Analysis
* Sales Forecasting
* AI Insights
* AI Business Recommendations
* Chat With Data
* PDF Report Generation

## Tech Stack

* Python
* Pandas
* NumPy
* Streamlit
* Scikit-Learn
* Google Gemini AI
* ReportLab
* Power BI
* SQL
